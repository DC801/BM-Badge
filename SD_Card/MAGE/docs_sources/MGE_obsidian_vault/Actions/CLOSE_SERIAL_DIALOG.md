# CLOSE_SERIAL_DIALOG

Ends any [[serial dialogs|serial dialog]] that is awaiting user input, such as a [[serial dialog options|free response question or a multiple choice question]].

## Example JSON

```json
{
  "action": "CLOSE_SERIAL_DIALOG"
}
```
