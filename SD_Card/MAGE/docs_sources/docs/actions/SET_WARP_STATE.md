# SET_WARP_STATE

Set the [warp state string](../scripts/variables#warp_state) to a specific value.

## Example JSON

```json
{
  "action": "SET_WARP_STATE",
  "string": "some kind of string"
}
```

## MGS Natlang

### Example

```mgs
script {
  set warp state to "some kind of string";
}
```

### Dictionary entry

```
set warp state (to) $string:string (;)
```
